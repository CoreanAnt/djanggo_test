위 내용을 바탕으로
db.php란 파일을 만들어 유지보수를 쉽게 만들자 (아이디 비번은 계속 치지 말자는 뜻)

<?php
$conn = mysqli_connect("localhost","root","123");
mysqli_select_db($conn, "test");
?>


이런식으로 만들었으면 같은 경로에 저장해주고
이제부터 include "db.php"; 이렇게 전처리를 해주자.
이후 data.php 란 파일을 만들어

<!DOCTYPE html>
<html>
<head>
    <title>데이터베이스 내용</title>
</head>
<body>
    <h1>MySQL 접속해서 데이터 가져오기</h1>

<?php
include "db.php";

$sql = "SELECT * FROM test ORDER BY id DESC";
$result = mysqli_query($conn, $sql);

 $dataCount = $result->num_rows;

    for ($i = 0; $i < $dataCount; $i++) {
      $memberInfo = $result->fetch_array(MYSQLI_NUM);
      echo "이름 : " . $memberInfo[0];
      echo "<br>";
      echo "아이디 : " . $memberInfo[1];
      echo "<br>";
      echo "직업 : " . $memberInfo[2];

    }
?>
</body>
</html>

이런식의 구조로 소스코드를 짜면 localhost/data.php 로 들어가면 데이터 베이스에 있는 데이터가 출력될 것이다.
